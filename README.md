# Expense Tracker Web App
An interactive web application to track and visualize expenses built with HTML, CSS and JavaScript.
# Features:
> Add, edit and categorize expenses
> Set reminders
> Dynamic charts for visualization
> Clean and intuitive UI for easy use
# Tools and Technologies :
> HTML 
> CSS 
> JavaScript
# How to run:
> Clone the respository : 'git clone https://github.com/Khadijat112/Expense-tracker.git'
> Open 'index.html' in your browser.
